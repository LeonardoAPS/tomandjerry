Tom Server:

Instructions

  • 1. Start the Node application (npm start or nodemon)
  • 2. Open your application and make sure Tom and Jerry Chrome extension is enabled
  • 3. Click on the extension button and enable the Edit mode
  • 4. From now on, every click on your browser will be captured by the Tom and Jerry. If you want to have detailed info, enable the Console view of your Chrome
  • 5. Once you're done, disable the Edit mode
  • 6. Go to the options of the T&J extension and make sure you have configured the server URL
    • 6a. Configure the Server URL as '[serverURL]/trigger/id=[ID]] - see below for the IDs
    • 6b. For Google Slides
    • Go to your GSlides presentation and click on File>Publish to web>Select Embed>Clibk on Publush
    • A code will be displayed, you have to collect the ID, which start with '2PACX-...'
    • Open another browser window with the URL '[serverURL]/gslides?id=[id]'
    • 6c. For Prezi
    • Go to Prezi and get the ID of the presentation identified in the URL
    • Open another browser window with the URL '[serverURL]/prezi?id=[id]'
  • 7. Starting doing your demo and the magic will hapen. Try to use a side-by-side view, on the left your presentation and on the right your demo application
